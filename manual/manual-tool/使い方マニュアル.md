# Manual Forge (マニュアル作成ツール) 使い方マニュアル

Manual Forge（マニュアル・フォージ）は、誰でも簡単に「インタラクティブなマニュアル」を作成できるツールです。
スクリーンショット画像を読み込み、その上に解説（ホットスポット）やページ移動ボタン（リンク）を配置して、配布可能なHTMLファイルとして保存できます。

## 1. ツールの起動方法

難しいインストール作業は不要です。

1.  配布されたフォルダ内の **`ManualForge_Tool.html`** を探します。
2.  ファイルを **ダブルクリック** してください。
3.  Webブラウザが立ち上がり、ツールが起動します。

## 2. マニュアルの作成手順

### ステップ 1: 画像の読み込み
画面中央にある「画像を選択」ボタンをクリックするか、マニュアルにしたい画像ファイル（スクリーンショットなど）を画面にドラッグ＆ドロップしてください。

### ステップ 2: 解説（ホットスポット）の追加
画像上の解説を入れたい場所を **クリック** すると、番号付きのマーカー（ホットスポット）が追加されます。

-   **移動**: マーカーをドラッグ＆ドロップで好きな場所に動かせます。
-   **編集**: マーカーをクリックすると、右側に編集画面が表示されます。
    -   **タイプ**: 「必須（赤）」「注意（黄）」「情報（青）」から重要度を選べます。
    -   **アイコン**: 数字や記号（● ! ?）に変更できます。
    -   **タイトル・説明文**: 表示されるメッセージを入力します。

![編集画面](/Users/nexpure/.gemini/antigravity/brain/275758ab-5955-415f-a071-d830db762c60/localized_ui_screenshot_1771471894355.png)

### ステップ 3: ページ移動ボタン（リンク）の追加
他のページへのリンクを作りたい場合は、画面左側のサイドバーにある **「リンクアイコン（🔗）」** をクリックします。

-   画面中央に「鎖マーク」のアイコンが現れます。
-   **編集**: リンクアイコンをクリックして、右側のパネルで設定します。
    -   **ラベル**: マウスを乗せた時に表示される文字（例：「次のページへ」）。
    -   **遷移先URL**: 移動先のファイル名（例: `step2.html`）やWebサイトのURLを入力します。

## 3. 保存と書き出し

### 作業の一時保存（プロジェクト保存）
作業を中断したい場合や、後で修正したい場合は、画面右上の **「保存 (JSON)」** をクリックしてください。
`manual-project.json` というファイルがダウンロードされます。
次回起動時に、サイドバーの設定アイコン（⚙️）からこのファイルを読み込むことで、続きから作業できます。

### 完成版の書き出し（HTML書出）
マニュアルが完成したら、画面右上の **「HTML書出」** をクリックしてください。
`manual-export.html` というファイルがダウンロードされます。

このHTMLファイルは単独で動作します。メールで送ったり、共有サーバーに置くだけで、誰でもあなたが作ったインタラクティブなマニュアルを閲覧できます。
